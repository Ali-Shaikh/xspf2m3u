= Xspf2m3u

*Author:* Pavel Argentov <argentoff@gmail.com>

== Usage:

    xspf2m3u help

    xspf2m3u -i infile.xspf -o outfile.m3u

This CLI program converts a playlist in XSPF format into Extended M3U playlist.
